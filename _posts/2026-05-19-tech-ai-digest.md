- The VC Corner (May 18): MCP unified Anthropic, OpenAI, Cursor, Microsoft, Vercel, Lovable in just 6 months after launch.

**Claude Code — "HTML is the new Markdown":**
- Anthropic engineers building with Claude Code are outputting HTML artifacts rather than Markdown — richer, interactive, web-native deliverables.
- Signals: (1) Claude Code evolving from text generation toward structured output; (2) HTML as a container for AI-generated interactive content; (3) Shift in how developer workflows are structured around AI outputs.

## Signals

1. **$12 trillion AI capex — Coatue's framing raises the stakes on the capex debate.** At $12T cumulative, AI infrastructure is being built at a scale that exceeds any prior technology buildout (internet was ~$300B–500B peak cycle). If the framework is right, the winner/loser split is extreme — concentrated value accrual in Nvidia, hyperscalers, and key application layer owners. The India angle: most of this capex is happening outside India. The Google Vizag $15B hub and Uber+Adani DC are India trying to capture a portion.

2. **Google I/O is the most significant competitive moment in AI this week.** If Google announces a credible Claude Code competitor ("super agent"), it would validate the developer-first AI infrastructure thesis and force Anthropic, Cursor, and GitHub Copilot to respond. A weak Google I/O announcement would reinforce Anthropic's current developer momentum. Watch this space.

3. **MCP skills layer as platform.** The speed of MCP adoption (6 months to 6+ major platforms) is unprecedented. The framing shift from "prompts as moat" to "skills as moat" has direct implications for India enterprise AI adoption: companies that package their workflows as MCP skills (like HDFC Bank's Agentic Mesh) are building platform-level moats, not just tool stacks.

4. **Samsung strike = AI supply chain tail risk.** HBM memory is already constrained; a 41,000-worker Samsung strike would worsen scarcity materially. Downstream effects: Nvidia delivery timelines slip → hyperscaler AI buildout delayed → potential relief on the AI capex overhang narrative. This is a low-probability, high-impact tail risk.

5. **"HTML is the new Markdown" signals Claude Code's product direction.** Moving from Markdown to HTML output means Claude Code is evolving toward web-native, interactive deliverables — potentially replacing parts of the frontend development workflow, not just backend code generation. For India: developer-facing tools like Replit, Vercel, and Lovable are already integrating Claude; Indian IT services firms (TCS, Infosys) need to track this for their own AI-augmented delivery models.

6. **Unitree IPO = robotics market signal.** A Chinese robotics company filing for IPO while profitable on hardware means the robotics market has crossed a commercial viability threshold. The $300M model layer bet (AI locomotion models) mirrors the foundation model investment logic — suggesting Unitree expects AI models to be a defensible differentiator in hardware too.

7. **Andreessen + Grey Swan converging on same thesis.** Two independent analytical voices in the same newsletter cycle making the same argument ("moat is not the model") is a signal, not coincidence. It reflects where the smart money is directionally thinking: AI commoditization is accepted, now the question is who captures value in the post-commodity layer. India fintech parallel: HDFC Bank's 100M customer dataset, Bajaj Finance's 119M franchise behavioral data, and Paytm's ₹6.5 lakh crore merchant GMV data are precisely the "what you're defending" for each institution.
